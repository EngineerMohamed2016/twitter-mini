# 🐦 Twitter Mini (Full-Stack Social Media App)

A full-stack social media application inspired by Twitter, built with modern Next.js architecture and scalable backend APIs.

---

## 🚀 Live Demo

https://twitter-mini-xyz.vercel.app

---

## 🛠️ Tech Stack

### 🎨 Frontend

* ⚛️ Next.js 16 (App Router)
* ⚡ React 19
* 🎨 Tailwind CSS

### 🔧 Backend

* 🟢 Node.js
* 🚀 Express.js (REST API)

### 🗄️ Database

* 🍃 MongoDB

---

## ✨ Features

* 🔐 Authentication & Authorization (JWT-based)
* 📝 Create & delete posts
* ❤️ Like / Unlike posts
* 💬 Comment system on posts
* 🔖 Bookmark posts (save & remove)
* 🔍 Search users by username
* 👤 User profile pages
* ⚡ Optimized rendering (Server & Client Components)
* 📱 Fully responsive UI

## ⚙️ Installation & Setup

### 📥 Clone the repo

```
git clone https://github.com/EngineerMohamed2016/twitter-mini
```

### 💻 Frontend Setup

```
cd client
npm install
npm run dev
```

### 🖥️ Backend Setup

```
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

### 🔧 Backend (.env)

```
PORT=5000
DB_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

### 🎨 Frontend (.env.local)

```
API_URL=http://localhost:5000/
NEXT_PUBLIC_SITE_URL = http://localhost:3000/
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
